# 📊 Customer Shopping Behaviour Analysis

🚀 End-to-End Data Analysis Project using **Python, SQL, and Power BI**

---

## 📌 Project Overview

This project analyzes **customer shopping behavior** to uncover insights on purchasing patterns, revenue trends, and customer segmentation.

The workflow follows a complete data pipeline:

> **Raw CSV → Python (Data Cleaning) → PostgreSQL (SQL Analysis) → Power BI (Dashboard)**

📁 Dataset: `customer_shopping_behavior.csv`  
📊 Records: **3,900 rows | 18 features**

---

## 🎯 Objectives

- Understand customer purchasing behavior
- Analyze revenue trends across demographics
- Identify high-value customers
- Evaluate impact of discounts & subscriptions
- Build an interactive business dashboard

---

## 🛠️ Tech Stack

- 🐍 **Python (Pandas, NumPy)** – Data cleaning & feature engineering  
- 🗄️ **PostgreSQL (SQL)** – Data analysis  
- 📊 **Power BI** – Dashboard & visualization  
- 🔗 **SQLAlchemy** – Data pipeline integration  

---

## 🔄 Project Workflow

1. **Data Preprocessing (Python)**
   - Handled missing values (review ratings)
   - Feature engineering (age groups, purchase frequency)
   - Column standardization

2. **Database Integration**
   - Loaded cleaned data into PostgreSQL

3. **SQL Analysis**
   - Solved **10 business questions**
   - Used:
     - Aggregations (`SUM`, `AVG`, `COUNT`)
     - Subqueries
     - CTEs
     - Window Functions (`ROW_NUMBER()`)

4. **Power BI Dashboard**
   - KPI Cards (Revenue, Customers, Avg Spend)
   - Revenue by Category & Age Group
   - Subscription Analysis
   - Interactive Filters

---

## 📊 Key Insights

- 💰 **Total Revenue:** ₹1.95 Cr  
- 👥 **79.9% customers are loyal** (11+ purchases)  
- 🎯 **Young Adults (18–31)** generate highest revenue (~26.7%)  
- 🛍️ **Discount users still spend above average**  
- 📉 Subscription users **do NOT spend more**  
- ⚠️ 72.5% repeat buyers are **not subscribed** → huge opportunity
